NEOVIM NAVIGATION

BASICS
h j k l        move cursor
w / b          next / previous word
0 / $          line start / end
gg / G         file start / end

EDIT
dd             delete line
yy             copy line
p / P          paste

SEARCH
/              search
n / N          next / prev match

BUFFERS
Shift+h        previous buffer
Shift+l        next buffer
Space f f      find files
Space f g      grep

CODE
gd             go to definition
K              hover docs
Space c f      format
